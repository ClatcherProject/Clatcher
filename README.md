Clatcher is an experimental open source social media project.

To setup this on your server, please follow the following steps

1. You need an Apache Webserver, with PHP and a MySQL-Database
2. Clatcher.sql is an empty MySQL-Dump to set the important relations. Load it into your database
3. Create an SQL user wo can SELECT, INSERT, UPDATE and DELETE the Clatcher database
4. If you don't know what are you doing, DON'T CHANGE ANYTHING AT THE CODE. If you want to change some metrics like filesize of uploaded files, look at php/config.inc.php.
